# CSDN 标注工具

一个用于生成 CSDN 专家标注草稿的本地辅助工具。工具会从 CSDN 文章 URL 或手动粘贴的 HTML 中提取正文内容，生成可人工审核的标注建议，支持片段标注和全文标注。

> 本工具只生成标注草稿，不会自动登录、提交或修改 CSDN 内容。最终选中文本、标注内容和提交操作仍需人工确认。

## 功能特点

- 支持输入 CSDN 文章 URL 自动抓取正文
- 支持粘贴网页 HTML，在网络抓取失败时继续使用
- 自动提取文章标题和正文段落
- 规则模式生成最多 5 条标注草稿
- 支持“全文标注”草稿生成
- 可选调用本机 Codex 优化标注内容
- 本地 Web 页面操作，无需部署服务器

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `csdn_annotation_ui.py` | 本地 Web UI 和接口服务入口 |
| `csdn_annotation_draft.py` | 文章解析、正文提取、规则标注生成逻辑 |
| `启动CSDN标注工具.bat` | Windows 一键启动脚本 |
| `网站地址.txt` | 本地访问地址记录 |

## 环境要求

- Windows 10/11
- Python 3.10 或更高版本
- 可选：本机已安装并登录 Codex CLI，用于“使用本机 Codex 优化草稿”功能

本项目目前只使用 Python 标准库，规则生成模式不需要额外安装依赖。

## 使用方法

### 方式一：双击启动

在 Windows 上双击：

```bat
启动CSDN标注工具.bat
```

启动后在浏览器打开：

```text
http://127.0.0.1:8765
```

### 方式二：命令行启动

```bash
python csdn_annotation_ui.py --host 127.0.0.1 --port 8765
```

然后访问：

```text
http://127.0.0.1:8765
```

## 页面操作流程

1. 输入 CSDN 文章 URL，或粘贴文章 HTML。
2. 选择生成数量，最多 5 条。
3. 如需全文标注，勾选“生成全文标注”。
4. 如需更自然的标注内容，勾选“使用本机 Codex 优化草稿”。
5. 点击“生成标注草稿”。
6. 复制单条或全部草稿，再到 CSDN 页面中人工确认和提交。

## Codex 增强模式

勾选“使用本机 Codex 优化草稿”时，工具会调用本机 `codex` 命令读取文章段落并返回 JSON 格式标注建议。

如果本机没有安装 Codex CLI，或 Codex 输出不符合格式要求，工具会自动回退到规则生成模式。

## 标注额度提醒

CSDN 专家标注通常存在额度限制：

- 每人每天最多 10 处
- 每篇文章最多 5 处
- 片段标注和全文标注共用额度
- 全文标注通常也按 1 处计算

请以 CSDN 页面实际规则为准。

## 注意事项

- 工具不会保存账号、Cookie 或登录信息。
- 使用 URL 抓取时可能受网络、证书、反爬策略影响；失败时建议复制网页 HTML 后粘贴到输入框。
- 生成内容只是草稿，提交前需要检查是否准确、客观、符合 CSDN 审核要求。
- `csdn_annotation_ui.log` 是本地运行日志，不建议提交到公开仓库。

## 命令行草稿生成

也可以直接使用命令行脚本生成草稿：

```bash
python csdn_annotation_draft.py "https://blog.csdn.net/..." --max 5 --format text
```

分析本地 HTML 文件：

```bash
python csdn_annotation_draft.py --html-file article.html --max 5 --format json
```
